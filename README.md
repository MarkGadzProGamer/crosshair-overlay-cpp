# 🎯 Crosshair Overlay Utility

**Language:** C++  
**Platform:** Windows (WinAPI)  
**Description:** A lightweight, high-performance overlay that provides a customizable crosshair for games.

### 🚀 Features:
- **Zero Latency:** Built with native Windows API for maximum performance.
- **Customization:** Change crosshair shape, size, and color.
- **Offset Support:** Move the crosshair to any position on the screen.
- **Always on Top:** Guaranteed visibility over game windows.

### 🛠 Tech Stack:
- WinAPI for window management.
- GDI+ for smooth rendering.
- Registry/Config saving (in progress).
